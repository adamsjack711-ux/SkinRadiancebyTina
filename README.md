# Tattoo Artist Website Template

A clean, single-page website template for clinical tattooing artists. Designed for medical and aesthetic tattooing services with a professional, high-end aesthetic.

## Sections

- Hero with call-to-action
- Artist bio and credentials
- Photo gallery
- Service pricing cards
- Contact / booking form (Formspree)

## Tech Stack

- HTML5 / Vanilla JavaScript
- Tailwind CSS (CDN)
- Google Fonts (Playfair Display, Inter)
- Formspree for form submission
